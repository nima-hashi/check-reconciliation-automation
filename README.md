# check-reconciliation-automation
Reconcile (match) checks against invoices automatically

Goal

Create a process that automatically:

Downloads paid check images from the bank.
Pulls important information from each check (vendor, amount, date, etc.).
Downloads vendor invoices from email.
Pulls important information from each invoice.
Matches invoices to checks that have been paid.
Flags missing invoices, missing checks, or amount mismatches.
Generates a report that can be used in Excel and/or QuickBooks.

The goal is to reduce manual work and make the reconciliation process faster and more accurate.
